# NPMame

## Run

Install required

- [Node.js/npm/npx](https://nodejs.org/en/download)

### npx

Start CLI

```bash
npx npmame
```

### npm

Install package globally

```bash
npm i -g npmame
```

Start CLI

```bash
npmame
```
